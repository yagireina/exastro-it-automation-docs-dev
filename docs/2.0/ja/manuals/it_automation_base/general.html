1. ログイン方法¶

1.1. はじめに¶

本書は、オーガナイゼーション作成直後の初回ログイン方法について記載したものです。

1.2. 初回ログイン¶

  1. 初回ログイン時は、必ず次のユーザ名とパスワードでログインしてください。
    表 1.37 初回ログイン時の入力内容¶

    ユーザ名またはメールアドレス

    パスワード

    adminまたはadmin@example.com

    password
  2. パスワードの更新を行い、送信ボタン を押下します。
  3. アカウント情報の更新し、 更新ボタン を押下します。
  4. 下記のようにWorkspace管理画面が表示されます。
